# AI Voice Board
### Голосовой ввод текста для Android

---

## Что это такое

**AI Voice Board** — клавиатура для Android, которая распознаёт вашу речь и автоматически вставляет текст в любое поле ввода: мессенджеры, заметки, почта, браузер и другие приложения.

Вы говорите — приложение печатает. Никакого ручного набора.

Распознавание работает через технологию OpenAI Whisper — одну из самых точных систем распознавания речи в мире. Поддерживается русский, английский и другие языки, а также смешанная речь.

## Установка

### Шаг 1. Установите приложение

Получите файл `AiVoiceBoard.apk` и установите его на телефон.

> Если Android спрашивает разрешение на установку из неизвестных источников — разрешите для вашего файлового менеджера.

### Шаг 2. Откройте приложение

Найдите **AI Voice Board** в списке приложений и откройте его.

### Шаг 3. Дайте разрешение на микрофон

Нажмите кнопку **Grant microphone permission** и разрешите доступ к микрофону.

### Шаг 4. Включите клавиатуру

Нажмите **Enable keyboard** — откроются системные настройки. Найдите **AI Voice Board** в списке и включите.

### Шаг 5. Выберите клавиатуру

Вернитесь в приложение и нажмите **Select AI Voice Board**. Выберите её в появившемся меню.

### Шаг 6. Введите API-ключ

В поле **API Key** введите ваш ключ доступа и нажмите **Save**.

## Как пользоваться

Откройте любое поле ввода в любом приложении. Внизу экрана появится панель AI Voice Board.

### Панель клавиатуры

```
[⌨]   Tap to record   [⌫] [🎤] [↵]
```

| Кнопка | Действие |
|--------|----------|
| 🎤 | Нажать — начать запись голоса |
| ⌫ | Удалить символ. Удерживать — быстрое удаление. Свайп вверх на **Clear** — очистить всё поле |
| ↵ | Перенос строки |
| ⌨ | Переключиться на другую клавиатуру |

### Запись голоса

1. Нажмите 🎤 — начнётся запись (таймер в центре)
2. Говорите
3. Нажмите **⏹** чтобы остановить и вставить текст
4. Нажмите **↑** чтобы остановить, вставить и сразу отправить сообщение
5. Нажмите **✕** чтобы отменить запись

### Во время распознавания

Пока идёт обработка, отображается анимация загрузки. Нажмите **✕** чтобы отменить.

### Меню (долгое нажатие на ⌨)

- **Retry last recording** — повторно отправить последнюю запись (если не было интернета или произошла ошибка)
- **Paste last text** — вставить последний распознанный текст ещё раз
- **Settings** — открыть настройки

## Настройки

| Параметр | Описание |
|----------|----------|
| **API Key** | Ключ доступа к сервису распознавания |
| **Model** | Модель распознавания (`gpt-4o-transcribe` — максимальное качество) |
| **Prompt** | Подсказка для модели: термины, стиль речи, язык |
| **Proxy** | Прокси-сервер (если требуется для подключения) |
| **Line break after sentence** | Автоматически ставить перенос строки после каждого предложения |

## Советы

- Говорите чётко, в обычном темпе — не нужно замедляться
- Приложение хорошо понимает смешанную русско-английскую речь
- Если не было интернета во время записи — восстановите подключение и нажмите **Retry** в меню
- Для быстрого переключения между AI Voice Board и обычной клавиатурой используйте кнопку ⌨
