# 🎙 AI Voice Board

Клавиатура голосового ввода для Android на базе OpenAI Whisper API.
Нажмите микрофон — скажите — текст вставится в любое поле ввода.

Дополнение к [ai-voice-input](https://github.com/HaiksonLab/ai-voice-input) (версия для Windows) — та же идея, нативно на Android в виде кастомной клавиатуры.

## Возможности

- Кастомная клавиатура (InputMethodService) — работает в любом приложении с полем ввода
- Распознавание речи через OpenAI Whisper API (отличное качество русского языка)
- Вставка текста в позицию курсора через `InputConnection`
- **Стоп + Отправить** — распознать и отправить сообщение одним нажатием (ChatGPT, Telegram и др.)
- **Отмена** во время записи или распознавания — прервать без вставки текста
- **Повтор** — переотправить последнюю запись, если запрос к API не удался (нет интернета, истёк ключ)
- **Вставить последнее** — вставить последний распознанный текст без повторной записи
- Backspace с удержанием для повтора и свайп-вверх на **Clear** для очистки поля
- Переключение на предыдущую клавиатуру одним нажатием (Yandex, Gboard и др.)
- Визуализация спектра (эквалайзер) во время записи
- Запись автоматически отменяется при скрытии клавиатуры; распознавание продолжается и вставляет текст только если клавиатура ещё открыта (иначе доступно через **Вставить последнее**)
- Проверка и установка обновлений прямо из приложения через GitHub Releases (бейдж + пункт меню)
- Опциональный перенос строки после каждого предложения
- Все параметры — в приложении, не нужно трогать код

## Требования

- Android 8.0+ (API 26)
- API-ключ OpenAI с доступом к Audio API: [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
- Интернет-соединение для распознавания

## Установка

**Вариант А — APK (рекомендуется):**
1. Скачайте `AiVoiceBoard-vX.X.X.apk` из [последнего релиза](https://github.com/HaiksonLab/ai-voice-board/releases/latest)
2. Скопируйте на телефон и установите (разрешите *установку из неизвестных источников*, если потребуется)
3. Откройте приложение **AI Voice Board**
4. Нажмите **Grant microphone permission** (разрешить микрофон)
5. Нажмите **Enable keyboard** → включите в системных настройках
6. Нажмите **Select AI Voice Board** → выберите как способ ввода
7. Введите свой API-ключ OpenAI и нажмите **Save**

**Вариант Б — сборка из исходников (требуется Android Studio / JDK 17):**
1. Скачайте или клонируйте репозиторий
2. Откройте в Android Studio или соберите из командной строки:
   ```bash
   ./gradlew assembleRelease
   # APK → app/build/outputs/apk/release/app-release.apk
   ```

## Настройка

Все параметры находятся на экране настроек приложения:

| Параметр | Описание | По умолчанию |
|---|---|---|
| API Key | API-ключ OpenAI | — |
| Model | Модель транскрипции | `gpt-4o-transcribe` |
| Prompt | Подсказка для модели (опционально) | технические термины |
| Proxy | SOCKS5 прокси для запросов к OpenAI API (опционально) | пусто |
| Line break after sentence | Перенос строки после `.!?` | вкл |

**Прокси** пример:
```
socks5h://127.0.0.1:1080
```

**Про модели:** актуальный список поддерживаемых моделей Whisper смотрите на [platform.openai.com/docs/models](https://platform.openai.com/docs/models).

**Про `Prompt`:** необязательный параметр. Помогает модели точнее распознавать профессиональные термины, имена, аббревиатуры. Укажите в нём ключевые слова из вашей области.

## Клавиатура

Клавиатура — одна строка с тремя состояниями: ожидание, запись и распознавание.

| Кнопка | Нажатие | Долгое нажатие |
|---|---|---|
| ⌨ | Переключиться на предыдущую клавиатуру | Меню: Повтор / Вставить последнее / Настройки / Проверить обновление — красная точка отмечает доступное обновление |
| 🎤 | Начать запись | — |
| ⌫ | Удалить один символ | Повтор; свайп вверх → **Clear** (очистить всё поле) |
| ↵ | Перенос строки | — |
| ✕ | Отменить запись / распознавание | — |
| ⏹ | Стоп → распознать → вставить | — |
| ↑ | Стоп → распознать → вставить → отправить (Enter) | — |

## Известные ограничения

### Галлюцинации при тишине

Если во время записи ничего не было сказано, а затем нажать кнопку остановки — API может вернуть не пустую строку, а произвольный текст: фрагмент из `Prompt` или тематически похожую фразу.

Это **ожидаемое поведение** Whisper и `gpt-4o-transcribe`: модели всегда пытаются декодировать что-то в текст и не имеют режима "вернуть пусто при отсутствии речи". Наличие `Prompt` усиливает эффект — модель достраивает текст из его контекста.

**Решение простое:** если вы начали запись и не сказали ничего — нажмите **✕** (отмена) вместо кнопки остановки. Отмена прерывает запись без обращения к API.

### Отправка в некоторых приложениях

Кнопка **↑** (Стоп + Отправить) использует действие, объявленное полем ввода (`IME_ACTION_SEND`). Большинство мессенджеров его учитывают. В **Telegram** отправка через действие клавиатуры работает только если включено *Настройки → Настройки чатов → Отправлять по Enter*.
